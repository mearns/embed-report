Specifies whether to save the report for every build or just the job (i.e., just the latest build).
project only
The report is only saved from the latest build, and is embedded in the top level status page of the job only.
build only
The report is saved for every build, and is embedded in the status page for the build only.
both project and build
The report is saved for every build, is embedded in the status page for each build, and the report from the latest build is embedded in the top level status page for the job as well.
